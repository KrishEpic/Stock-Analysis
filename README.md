# Stock-Analysis
This project analyzes historical stock price data using statistical tools. It focuses on understanding trends, return behavior, and volatility. This analysis is implemented in Python inside a Google Colab notebook, and uses real market data from Yahoo Finances

# Features 
1. Downloading historical price data:
  Using the yfinance library to pull:
  * Open
  * High
  * Low
  * Close
  * Volume
2. Moving Averages:
  Using the 20 day and 50 day moving averages to look at short term and long term trends.
3. Daily Returns:
  Calculated the daily returns to show the percentage change in the stock
4. Volatility:
  Calculated the volatility using the standard deviation of the returns, to show how much a stock moves on average.
5. Visualization:
  Using Matplotlib to creatre charts that show:
  * Closing price over time
  * 20-day and 50-day moving averages
  * Daily return
